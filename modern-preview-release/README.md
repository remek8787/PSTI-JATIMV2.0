# PSTI Jatim Modern Preview Release

Checkpoint release untuk preview modern PSTI Jatim.

Isi utama:
- public portal modern
- blog/media center
- tracer atlet/pelatih
- anggota: pengurus, klub, wasit
- kompetisi: kejurda, piala gubernur, agenda/jadwal
- admin dashboard sederhana
- tutorial admin
- PWA installable support

Catatan keamanan:
- `data/users.json` tidak disertakan di repo.
- Gunakan `data/users.example.json` sebagai template dan buat password hash baru di server.
- Jangan commit credential hosting/admin/API key.

Deploy preview saat ini: `https://pstijatim.com/modern-preview/`
